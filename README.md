# 🔐 密码生成器

一个安全、快速、可自定义的在线密码生成工具。

## ✨ 特性

- 🔒 **安全可靠**：使用 Web Crypto API 生成加密安全的随机密码
- ⚡ **快速生成**：毫秒级响应，支持实时预览
- 🎨 **现代 UI**：简洁美观的用户界面，响应式设计
- ⚙️ **高度可定制**：支持自定义长度、字符类型、排除选项
- 📊 **强度检测**：实时显示密码强度和安全性建议
- 📋 **一键复制**：快速复制生成的密码
- 🌐 **完全离线**：所有密码在浏览器本地生成，保护隐私

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 📋 功能列表

### 核心功能（已实现）

- ✅ 基础密码生成
- ✅ 长度设置（4-128 字符）
- ✅ 字符类型选择（大写、小写、数字、特殊字符）
- ✅ 排除相似字符选项
- ✅ 排除歧义字符选项
- ✅ 密码强度检测和评分
- ✅ 复制到剪贴板
- ✅ 响应式设计

### 计划功能

- ⏳ 密码类型预设
- ⏳ 批量生成
- ⏳ 密码历史记录
- ⏳ 密码分析（熵值、破解时间）
- ⏳ 导出功能
- ⏳ 密码短语生成

## 🛠️ 技术栈

- **React 18** - UI 框架
- **Vite** - 构建工具
- **Web Crypto API** - 加密安全的随机数生成
- **CSS3** - 样式和动画
